
=======================================
Aalto Scientific Computing - Admin Docs
=======================================

This site contains documentation about maintaining scientific and 
data-intensive computing at Aalto.  It is targeted towards Aalto 
Science IT admins, but has some useful information for other people 
interested in cluster maintenance.

These docs are maintained by `Aalto Science-IT
<http://science-it.aalto.fi>`_.
